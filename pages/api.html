Lobbyist Lookup API

Usage

  • Load /api/ with parameters
    Parameter Description
    surname Lobbyist surname
    organization Organization name
    client Client name


  • Results are returned in a JSON array of matching lobbyist organization registration informations.
      Example:
    • /api/?surname=gleason&organization=cyber security
    • /api/?client=pharm
